Parsing Words
expect	( address length -- )			Reads characters from the current input device.
>body	( header -- body )			Returns the address of the body from a given header.
find	( address length -- header )		Searches dictionary for matching string.
number	( address length -- value flag )	Attempts to convert the string into a number.
word	( delimiter -- address length )		Identifies words in the input buffer.

Flow Control Words
branch	( -- )		Branches unconditionally to the adderss in the following cell.
0branch	( flag -- )	Branches conditionally, if flag is FALSE, othewise take no action.

Built-in Constants
here	( -- address )	Returns the address of the start of free memory.
pad	( -- address )	Returns the address for use as a temporary pad or work area.
r0	( -- address )	Beginning address of return stack.
s0	( -- address )	Beginning address of data stack

Built-in Variables
>in	( -- address )	Current read position in input buffer.
latest	( -- address )	Latest dictionary entry.
state	( -- address )	Whether in interpret or compile mode.
rp	( -- address )	The address that holds the address of the top of the return stack.
sp	( -- address )	The address that holds the address of the top ot the data stack.

Compilation Words
[		( -- )			To interpret mode.
]		( -- )			To compile mode.
'		( "<name>" -- header )	Returns header of the following word.
[']		( "<name>" -- header )	Compiles following word as a literal.
,		( n -- )		Compiles a cell on the stack.
c,		( n -- )		Compiles lower 8-bits of the cell on the stack.
align		( -- )			Aligns HERE to the next aligned address.
allot		( n -- )		Allot *n* bytes by advancing HERE by *n* bytes.
compile		( -- )			Compiles following word.
execute		( header -- )		Executes the word.
header		( address length -- )	Create a new dictionary header.
create		( -- )			Create a new dictionary header.				: create parse header ;
hidden		( header -- )		Toggles hidden flag in the header.			: hidden >count dup c@ FLAG_HIDDEN xor swap c! ;
immediate	( -- )			Togglest hidden flag in the header of latest word.	: immediate latest >count dup c@ FLAG_IMMEDIATE xor swap c! ; immediate
lit		( -- n )
literal		( n -- )		Compiles LIT and *n* into current definition.		: literal ['] lit compile compile ;
